# Fuse Importer

The Fuse Importer is a powerful tool for importing data into your application. This guide will help you understand how to use and customize the Fuse Importer.

## Basic Usage

To use the Fuse Importer, you first need to create an instance:

```typescript
const importer = new FuseImporter(apiToken, templateSlug, options);
```

## Customizing Importer Styles

The Fuse Importer supports extensive style customization to match your application's design. You can modify colors, fonts, and other visual aspects using the `setCustomStyles` method.

For detailed information on customizing styles, including examples and best practices, please refer to our [Customizing Styles](/customizing-styles) guide.
